ECMAScript 6

(o ES2015)

La sesta edizione, conosciuta come ECMAScript 2015[9] ma anche come "Harmony", apporta modifiche sintattiche significative che aprono la porta ad applicazioni più complesse, includendo classi e moduli definendoli semanticamente come lo "strict mode" in ECMAScript 5. Altre nuove features sono diversi iteratori, for/of loop, Python-style generators e generator expression, arrow functions, binary data, typed arrays, collections (maps, sets e weak maps), promises, miglioramenti delle funzioni matematiche, reflection e proxies (metaprogrammazione per virtual objects). Arrow functions (II)

Prima: 

var myAwesomeObject = {
  results = [],
  getResults = function() {
    var self = this;
    myApiClient.getCustomers(function(customers) {
       self.results = customers;
    });
  }
};
Dopo: 

var myAwesomeObject = {
  results = [],
  getResults = () => myApiClient.getCustomers(customers => {
    this.results = customers;
  });
};

Default parameters

Prima 

              function sayHello(greeter, name) {
                greeter = greeter || 'Hello ';
                name = name || ' John Doe';

                console.log(greeter, name); // Hello John Doe
              }
Dopo 

                function sayHello(greeter = 'Hello ', name = 'John Doe') {
                  console.log(greeter, name); // Hello John Doe
                }

Classi (I)


            class Car {
              constructor(model) {
                this.model = model;
              }

              sayModel() {
                console.log(`I'm a ${this.model}`);
              }
            }

            const clio = new Car("Clio");
            clio.sayModel(); // I'm a Clio

Classi (II)


              class Clio extends Car {
                constructor() {
                  super("Clio");
                }
              }
  
              const myClio = new Clio();
              myClio.sayModel(); // I'm a Clio

Async/Await (I)


            function getUserFromApis () {
              return new Promise((resolve, reject) => {
                request('https://mybackend.com/user/666', (err, res, body) => {
                  if (err) {
                    reject(err); return;
                  }
                  resolve(body);
                });
              });
            }

Async/Await (II)


            function sayMyName () {
              getUserFromApis()
                .then(function(user) {
                  console.log(user);
                })
                .catch(function(error) {
                  console.warn(error);
                });
            }

Async/Await (III)


            const getUserFromApis = () => {
              return request('https://mybackend.com/user/666');
            }

            const sayMyName = async () => {
              try {
                const user = await getUserFromApis();
                console.log(user);
              } catch() {
                console.warn(error);
              }
            }
